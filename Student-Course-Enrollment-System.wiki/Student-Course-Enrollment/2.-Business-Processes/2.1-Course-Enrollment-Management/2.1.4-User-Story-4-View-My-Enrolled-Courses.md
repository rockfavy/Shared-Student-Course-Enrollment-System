# User Story 4 — View My Enrolled Courses

---

## Use Case: View My Enrolled Courses

::: mermaid
sequenceDiagram
    participant Student
    participant Frontend as Blazor WASM
    participant API as Minimal API
    participant EnrollmentService
    participant DB as In-Memory DB

    Student->>Frontend: Navigate to "My Enrollments"
    Frontend->>API: GET /enrollments/me
    API->>EnrollmentService: GetStudentEnrollmentsAsync(studentId)
    EnrollmentService->>DB: Query enrollments with courses
    DB->>EnrollmentService: Return enrollments with course details
    EnrollmentService->>API: Return enrollments
    API->>Frontend: Return enrollments JSON
    Frontend->>Student: Display enrolled courses
:::

**User Story:** As a student, I want to view all the courses I am enrolled in so I can track my studies.

**Actors:** Student, System

**Description / Goal:**  
Student views a list of all courses they are enrolled in, including course details and enrollment date. This allows the student to track their course load and manage enrollments.

**Preconditions:** Student is authenticated.

**Main Flow:**  
1. Student navigates to "My Enrollments" page  
2. System retrieves student's enrollments from API  
3. System displays enrolled courses with details  
4. Student can see all enrolled courses  
5. Student can view course details  
6. Student can deregister from courses  

**Alternate Flows / Exceptions:**  
- No enrollments → system displays "You haven't enrolled in any courses yet" message  
- Student not authenticated → system redirects to login  
- API error → system displays error message  

**Postconditions:**  
- Student can see their enrolled courses  
- Student can manage enrollments
