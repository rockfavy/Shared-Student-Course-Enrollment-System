# 3. Workflow Diagrams

Below are workflows covering *every screen* presented in the system.

---

## 3.1 Registration Workflow

:::mermaid
sequenceDiagram
    User->>RegistrationPage: Open registration screen
    User->>RegistrationPage: Enter registration details
    RegistrationPage->>AuthClient: Submit registration
    AuthClient->>API: POST /auth/register
    API->>DB: Create student account
    DB-->>API: Return success
    API-->>AuthClient: Return auth token
    AuthClient-->>RegistrationPage: Registration success
    RegistrationPage->>CoursesPage: Redirect to courses
:::

---

## 3.2 Login Workflow

:::mermaid
sequenceDiagram
    User->>LoginPage: Open login screen
    User->>LoginPage: Enter credentials
    LoginPage->>AuthClient: Submit login
    AuthClient->>API: POST /auth/login
    API->>DB: Validate credentials
    DB-->>API: Return student data
    API-->>AuthClient: Return auth token
    AuthClient-->>LoginPage: Login success
    LoginPage->>HomePage: Redirect to home
:::

---

## 3.3 Browse Courses Workflow

:::mermaid
sequenceDiagram
    Student->>CoursesPage: Load courses page
    CoursesPage->>CoursesClient: GET /courses
    CoursesClient->>API: Request courses list
    API->>DB: Query courses
    DB-->>API: Return courses
    API-->>CoursesClient: Return courses data
    CoursesClient-->>CoursesPage: Display courses
:::

---

## 3.4 Course Details Workflow

:::mermaid
sequenceDiagram
    Student->>CoursesPage: Click course
    CoursesPage->>CourseDetailsPage: Navigate to details
    CourseDetailsPage->>CoursesClient: GET /courses/{id}
    CoursesClient->>API: Request course details
    API->>DB: Query course
    DB-->>API: Return course data
    API-->>CoursesClient: Return course details
    CoursesClient-->>CourseDetailsPage: Display course
:::

---

## 3.5 Enroll in Course Workflow

:::mermaid
sequenceDiagram
    Student->>CourseDetailsPage: Click enroll
    CourseDetailsPage->>EnrollmentsClient: POST /enrollments
    EnrollmentsClient->>API: Create enrollment
    API->>DB: Validate and create enrollment
    DB-->>API: Return success
    API-->>EnrollmentsClient: Confirm enrollment
    EnrollmentsClient-->>CourseDetailsPage: Show confirmation
    CourseDetailsPage->>MyEnrollmentsPage: Redirect to enrollments
:::

---

## 3.6 View My Enrollments Workflow

:::mermaid
sequenceDiagram
    Student->>MyEnrollmentsPage: Load enrollments page
    MyEnrollmentsPage->>EnrollmentsClient: GET /enrollments/me
    EnrollmentsClient->>API: Request student enrollments
    API->>DB: Query enrollments
    DB-->>API: Return enrollments
    API-->>EnrollmentsClient: Return enrollment data
    EnrollmentsClient-->>MyEnrollmentsPage: Display enrollments
:::

---

## 3.7 Deregister from Course Workflow

:::mermaid
sequenceDiagram
    Student->>MyEnrollmentsPage: Click deregister
    MyEnrollmentsPage->>EnrollmentsClient: DELETE /enrollments/{id}
    EnrollmentsClient->>API: Remove enrollment
    API->>DB: Delete enrollment record
    DB-->>API: Return success
    API-->>EnrollmentsClient: Confirm removal
    EnrollmentsClient-->>MyEnrollmentsPage: Show confirmation
    MyEnrollmentsPage->>MyEnrollmentsPage: Refresh list
:::
