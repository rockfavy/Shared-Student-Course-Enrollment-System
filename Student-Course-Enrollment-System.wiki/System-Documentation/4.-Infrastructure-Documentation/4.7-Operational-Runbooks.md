# 2. Common Tasks

## 2.1 Application Startup

1. Navigate to project directory
2. Restore dependencies: `dotnet restore`
3. Start API: `dotnet run --project src/StudentCourseEnrollment.Api`
4. Start Frontend: `dotnet run --project src/StudentCourseEnrollment.Frontend`

## 2.2 Database Reset

1. Stop application
2. Restart application
3. Database is automatically recreated

## 2.3 Troubleshooting

### Application Won't Start
- Check .NET SDK version
- Verify dependencies restored
- Check configuration files

### API Not Responding
- Verify API is running
- Check port configuration
- Review error logs

### Frontend Can't Connect to API
- Verify API URL in configuration
- Check CORS settings
- Verify API is running
